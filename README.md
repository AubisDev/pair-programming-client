<!-- GETTING STARTED -->
## Getting Started
  ```sh
  Clone the repository
  ```
 
* How to install 

  ```
   cd programming-rooms-client
  ```
  
* yarn
  ```sh
  yarn install
  //or
  pnpm install 
  ```
  
  ```sh
  yarn dev
  //or
  pnpm dev
  ```
